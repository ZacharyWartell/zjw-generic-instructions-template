The instructions in this assignment assume you have already setup Visual Studio Code and your ITCS_3120 directory, ~/ITCS_3120 as described in the earlier assignment ITCS 3120 : Visual Studio Code Tutorial
  1. New Repo  Create a new remote repository on the Git server called "itcs3120-t5-webgl-tutorial-1".
    Follow all the standard practices for the course, (see https://webpages.charlotte.edu/zwartell/Teaching/Git%20Tutorial/Git%20Tutorial.html#How_to_Create_New_Remote_Repository)
  2. Clone  Clone your repo into your course directory
    lucretius@Monad42 ~/ITCS_3120/ $git clone URL_To_Your_Repository/3120-t5-webgl-tutorial.git
  3. VS Code Project Setup  Keep one window and instance of VS Code open to the WebGL_Programming_Guide directory.  Now you will open a second window and instance of VSCode for your own code.     During this tutorial keep both of these two instances of VSCode open.
